# Kanixian mod (wip)

* library : https://github.com/kitao/pyxel

## 概要
- fork して改変することでプログラミングを学習中

## 遊び方
- wasmのリンクからの起動　キーボード (<> + Space) でのみ操作可能　（gamepad で操作できない理由は不明）
- .py ファイルでの起動　ゲームパッド(anaglog-Stick / Dpad +  A / B) でも　操作可能
